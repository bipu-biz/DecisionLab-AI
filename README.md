## 🔥 Concept

Solve real-world backend scenarios →  
Submit your approach →  
Get instant AI-powered evaluation with structured feedback.

---

## 🧠 What Makes This Different?

Most projects:
- ❌ Static forms  
- ❌ Basic CRUD  
- ❌ Simple chatbots  

This project:
- ✅ Simulates real interview scenarios  
- ✅ Uses AI to evaluate reasoning (not keywords)  
- ✅ Provides structured feedback like a human reviewer  

---

## ✨ Features

- 🔐 Authentication (Login/Register)
- 🛡️ Protected Routes
- 📊 Scenario-Based Questions
- 🤖 AI Answer Evaluation
- 📈 Detailed Feedback:
  - Score (/10)
  - Clarity
  - Correctness
  - Depth
  - Improvements
- 🎨 Clean UI with Tailwind CSS

---

## 🧠 How AI Works

Instead of keyword matching, the AI:

- Understands the meaning of your answer  
- Evaluates reasoning and approach  
- Assigns structured scores  
- Generates improvement suggestions  

> 💡 Acts like a real interviewer reviewing your answer.

## ⚙️ Setup Instructions

### 1. Clone Repository

git clone https://github.com/bipu-biz/DecisionLab-AI  
cd DecisionLab-AI

---

### 2. Backend Setup

cd backend  
npm install  

Create `.env` file:

PORT=5000  
MONGO_URI=your_mongo_uri  
JWT_SECRET=your_secret  
AI_API_KEY=your_api_key  

Run backend:

npm run dev  

---

### 3. Frontend Setup

cd frontend  
npm install  
npm run dev
